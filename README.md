# بلا حدود للاستقدام — Landing Page

Static marketing site (single `index.html` + `assets/`). No build step. Arabic-RTL.
Deployed via GitHub Pages; custom domain mapped from Hostinger DNS.

- Edit `index.html`, commit, push → GitHub Pages auto-deploys.
- Images in `assets/img/` are licensed-free stock **placeholders** — replace with the office's own photos.
- `.nojekyll` keeps Pages from processing the site with Jekyll.
